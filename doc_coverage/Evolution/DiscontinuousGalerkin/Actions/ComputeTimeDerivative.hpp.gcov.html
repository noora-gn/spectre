      41             :  * \brief Computes the time derivative for a DG time step.
      42             :  *
      43             :  * Computes the volume fluxes, the divergence of the fluxes and all additional
      44             :  * interior contributions to the time derivatives (both nonconservative products
      45             :  * and source terms). The internal mortar data is also computed.
      46             :  *
      47             :  * The general first-order hyperbolic evolution equation solved for conservative
      48             :  * systems is:
      49             :  *
      50             :  * \f{align*}{
      51             :  * \frac{\partial u_\alpha}{\partial \hat{t}}
      52             :  *  + \partial_{i}
      53             :  *   \left(F^i_\alpha - v^i_g u_\alpha\right)
      54             :  *   = S_\alpha-u_\alpha\partial_i v^i_g,
      55             :  * \f}
      56             :  *
      57             :  * where \f$F^i_{\alpha}\f$ are the fluxes when the mesh isn't moving,
      58             :  * \f$v^i_g\f$ is the velocity of the mesh, \f$u_{\alpha}\f$ are the evolved
      59             :  * variables, \f$S_{\alpha}\f$ are the source terms, and \f$\hat{t}\f$ is the
      60             :  * time in the logical frame. For evolution equations that do not have any
      61             :  * fluxes and only nonconservative products we evolve:
      62             :  *
      63             :  * \f{align*}{
      64             :  * \frac{\partial u_\alpha}{\partial \hat{t}}
      65             :  *   +\left(B^i_{\alpha\beta}-v^i_g \delta_{\alpha\beta}
      66             :  *   \right)\partial_{i}u_\beta = S_\alpha.
      67             :  * \f}
      68             :  *
      69             :  * Finally, for equations with both conservative terms and nonconservative
      70             :  * products we use:
      71             :  *
      72             :  * \f{align*}{
      73             :  * \frac{\partial u_\alpha}{\partial \hat{t}}
      74             :  *   + \partial_{i}
      75             :  *   \left(F^i_\alpha - v^i_g u_\alpha\right)
      76             :  *   +B^i_{\alpha\beta}\partial_{i}u_\beta
      77             :  *   = S_\alpha-u_\alpha\partial_i v^i_g,
      78             :  * \f}
      79             :  *
      80             :  * where \f$B^i_{\alpha\beta}\f$ is the matrix for the nonconservative products.
      81             :  *
      82             :  * The mesh velocity is added to the flux automatically if the mesh is moving.
      83             :  * That is,
      84             :  *
      85             :  * \f{align*}{
      86             :  *  F^i_{\alpha}\to F^i_{\alpha}-v^i_{g} u_{\alpha}
      87             :  * \f}
      88             :  *
      89             :  * The source terms are also altered automatically by adding:
      90             :  *
      91             :  * \f{align*}{
      92             :  *  -u_\alpha \partial_i v^i_g,
      93             :  * \f}
      94             :  *
      95             :  * For systems with equations that only contain nonconservative products, the
      96             :  * following mesh velocity is automatically added to the time derivative:
